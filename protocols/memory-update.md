# Protocol: Memory Update

Promote only durable, reusable knowledge into memory.

## Memory targets

- `memory/references/` — systems, papers, repos, products, benchmarks
- `memory/entities/` — companies, labs, authors, tools, datasets
- `memory/claims/` — reusable claims with source pointers
- `memory/patterns/` — architecture patterns and anti-patterns
- `memory/benchmarks/` — evaluation tasks and rubrics

## Rule

Do not promote weak or unverified claims into memory without marking confidence.
