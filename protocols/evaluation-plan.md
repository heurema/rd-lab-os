# Protocol: Evaluation Plan

## Scorecard dimensions

- Coverage
- Source quality
- Citation accuracy
- Contradiction handling
- Actionability
- Reproducibility
- Evidence density
- Freshness disclosure

## Internal scoring scale

Use 0–3 for each dimension:

- 0: absent
- 1: weak
- 2: acceptable
- 3: strong

## Rule

Evaluate the artifact chain, not only the final prose.
