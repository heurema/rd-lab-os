# Protocol: Question Graph

The question graph turns a topic into an investigation map.

## Required sections

- Root question
- Sub-questions
- Alternative framings
- Stakeholders
- Schools of thought
- Likely controversies
- Evidence gaps
- Follow-up branches
- Kill criteria / what would change the conclusion

## Rule

A question graph is not a keyword list. It is a map of what must be known before a decision-grade answer is possible.
