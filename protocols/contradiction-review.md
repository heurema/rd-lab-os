# Protocol: Contradiction Review

Use this file to preserve uncertainty instead of smoothing it away.

## Required checks

- Unsupported claims
- Conflicting sources
- Stale evidence
- Missing primary sources
- Missing counterarguments
- Suspicious consensus
- Claims that rely only on social signals
- Claims that rely only on vendor/product claims

## Output

Group findings by severity:

- Critical
- Medium
- Minor
- Needs follow-up search
