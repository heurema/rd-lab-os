# Operating Model

## Verdict

Build a federated R&D Lab OS, not a single super-agent.

## Worker roles

| Worker | Best role |
|---|---|
| ChatGPT Deep Research | Long-horizon general investigator and synthesis partner |
| Gemini Deep Research | Editable-plan general investigator and broad web/Workspace scan |
| Codex | Builder, repo inspector, parser writer, normalizer, validator |
| Claude Code | Adversarial reviewer, architecture critic, contradiction hunter |
| Gemini CLI | Long-context reader, bulk scanner, local-file analyst |
| Elicit / ScholarQA / PaperQA2-like tools | Scientific-literature specialist |
| Perplexity | Fast scout and source discovery |
| last30days-style tools | Recency, social, market, and attention-signal collector |

## Core unit

The unit of work is a research run folder, not a chat.

## Source of truth

The evidence ledger is the source of truth. The final memo is a derivative artifact.
