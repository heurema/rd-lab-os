# Worker Roles

## Research Director

Defines decision context, scope, and acceptance criteria.

## Source Scout

Finds sources, classifies source modes, and ranks quality.

## Evidence Extractor

Turns sources into claim-level ledger rows.

## Synthesizer

Writes structured synthesis from ledger rows only.

## Critic

Finds unsupported claims, weak sources, contradictions, stale data, and overconfident recommendations.

## Memory Curator

Promotes reusable claims, references, patterns, and decisions into `memory/`.
