# Quality Gates

## Run validity gate

A run is not complete unless all required artifacts exist.

## Evidence gate

The final memo should contain no major strategic claim that is absent from `03_evidence_ledger.csv`.

## Source quality gate

For technical/scientific topics, prefer academic papers, official docs, code repos, benchmark pages, and primary product docs.

## Contradiction gate

If sources disagree, preserve the disagreement. Do not average it away.

## Provenance gate

Every source must have a `source_mode`.

## Decision gate

Every recommendation should map to at least one of:

- a decision;
- an experiment;
- a risk;
- a next research question.
