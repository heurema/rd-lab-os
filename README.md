# R&D Lab OS

An orchestration-first operating system for maximally deep research using existing frontier-agent subscriptions and coding agents.

This repo is **not** a model-training project. It is a repeatable research workflow with Codex-compatible skills, protocol templates, evidence ledgers, critic checklists, and durable memory artifacts.

## Core workflow

```text
Research Spec
→ Question Graph
→ Source Map
→ Evidence Ledger
→ Contradiction Review
→ Synthesis
→ Critic Review
→ Final Memo
→ Decision Log / Memory Update
```

## First command

```bash
python3 scripts/new_run.py "deep research agents reference atlas"
```

Then fill the artifacts in the created folder under `runs/`.

## Important rule

The final memo is not the source of truth. The evidence ledger is.
